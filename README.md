# Financial-Performance-Analysis-Project
Financial Performance Analysis using Python and Power BI with interactive dashboard and key business insights.

📊 Financial Performance Analysis using Python & Microsoft Power BI 📊

📝 Introduction:

Understanding financial performance is crucial for businesses to make informed decisions. This project focuses on analyzing sales, profit, and overall business performance using financial data. By leveraging Python for data cleaning and Microsoft Power BI for visualization, this project provides meaningful insights into revenue trends, profitability, and business segments.

🎯 Project Overview:

The Financial Performance Analysis project aims to evaluate business performance by analyzing key financial metrics such as revenue, profit, and profit margin. The dashboard helps stakeholders understand business trends, identify high-performing areas, and make data-driven decisions.

❗ Problem Statement:

Businesses often struggle to:

Track revenue and profitability effectively

Identify top-performing products and regions

Understand the impact of discounts on profit

Analyze monthly and segment-wise performance

This project solves these problems by creating an interactive dashboard that highlights:

Revenue trends

Profitability analysis

Segment performance

Product-level insights

📂 Dataset Information:

The dataset contains financial transaction data with key attributes:

📌 Data Definition:

Segment: Customer segment (Government, Midmarket, etc.)

Country: Country where sales occurred

Product: Product name

Discount Band: Discount category

Units Sold: Number of units sold

Manufacturing Price: Cost of production

Sale Price: Selling price per unit

Gross Sales: Total sales before discount

Discounts: Discount amount

Sales: Net sales after discount

COGS: Cost of goods sold

Profit: Net profit

Date: Transaction date

Month Number: Month (1–12)

Month Name: Month name

Year: Year
